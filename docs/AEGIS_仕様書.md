| 担当 | 作業 |

## 【対応会場】

徳山 / 大村 / 下関 / 若松 / 蒲郡 / 唐津 / 芦屋（7会場）

それ以外の会場が入力された場合はエンジンはabortを返す。

## 【主要出力指標の定義】

### confidence（展開信頼度）
```
confidence = min(|dominant_score| / 3.0, 1.0) × min(score_gap / 1.5, 1.0)
```
- **読み方：** 0.7以上 = 展開が確信に近い / 0.3以下 = 混戦

### coverage
```
coverage = 最大8点の確率の合計（%）
```

| 展開パターン | 目安 |
|---|---|
| 逃げ確信（dominant=逃げ かつ confidence≥0.7） | 30%以上 |
| まくり / まくり差し | 15%以上 |
| それ以外（混戦・差し） | 20%以上 |

## 【女子補正ルール】

| レースタイプ | 条件 | 補正 |
|------------|------|------|
| 全男子戦 | is_female=true が 0人 | 補正なし |
| **混合戦** | **is_female=true が 1～5人** | **該当女子艇のみ補正適用** |
| 女子戦 | is_female=true が 6人 | 補正なし |

**補正内容（混合戦の女子艇のみ）：**
- win_rate × 0.90
- local_rate × 0.90
- series_st_avg + 0.020秒（上限0.30）
- exh_st + 0.020秒（上限0.30）

## 【期待度判定】

```
高: confidence >= 0.7  かつ  coverage_warn == "OK"
低: confidence <  0.3  かつ  coverage_warn == "⚠coverage目安未達"
中: 上記いずれにも該当しない場合
```

### 推奨アクション
```
高 → 「3連単で勝負を推奨」
中 → 「3連複への切替を推奨（着順まで絞り切れていない可能性）」
低 → 「見送りを推奨（混戦・荒れ要素が強い）」
```
