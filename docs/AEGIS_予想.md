# AEGIS V5.1 予想出力プロンプト（STEP3）

## 概要

AEGIS V5.1 エンジンの計算結果を、見やすい3連単・3連複フォーマットに変換して表示するステップです。

## 出力フォーマット

### 基本構成

```
═══════════════════════════════════════════
AEGIS V5.1　{会場} {レース番号}R
───────────────────────────────────────────
【環境】{展開パターン}（gap={スコア差} / conf={信頼度}）

【3連単買い目】10点
① {組}  {確率}%  {◎○▲}
② ...

【期待度】{高/中/低}
推奨: {推奨アクション}

【チェック】
（詳細情報）
═══════════════════════════════════════════
```

### 3連単フォーマット

**買い目構成：**
- 本命2点
- 抑え2点
- 頭違い2点
- 保険3-4着候補2点
- 保険5-6着候補2点

**計10点固定**

### 3連複フォーマット（参考）

- 本命保険の不足分を4点で補填
- 着順を無視した3艇の組み合わせ

## 出力内容

### 環境情報

| 項目 | 説明 |
|------|------|
| dominant | 展開パターン（逃げ/差し/まくり等） |
| score_gap | スコア差分 |
| confidence | 展開信頼度（0.0～1.0） |
| coverage | カバー率（%） |

### 期待度判定

```
高: confidence ≥ 0.7  かつ  coverage ≥ 目安
中: その他
低: confidence < 0.3  かつ  coverage < 目安
```

### スコアランキング

全6艇を以下の指標でランキング表示：
- head（頭スコア）
- 2nd（2着スコア）
- 3rd（3着スコア）
- blended_st（ブレンド ST）

## 注意事項

- 期待度は投資判定の参考情報であり、的中を保証するものではない
- 「低」判定でも、ユーザーから明示的に買い目の提示を求められた場合は通常通り提示する
- サンプル数が少ない段階での運用であることを踏まえ、ユーザーから求められた場合は実績を確認した上で言及する
